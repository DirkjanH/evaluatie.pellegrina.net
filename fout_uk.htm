Form sending problems

If you encounter problems when sending in a form, it might have to do with a firewall or anti-virus program, which blocks part of the information the form script needs to process the form data. You will then see an error message, reading (in Dutch):

Fout tijdens mail versturen

Het versturen van mail is mislukt. Uw browser stuurt geen Referer: header. Dit script werkt niet zonder de Referer header. Pas de browser instellingen aan en probeer het nogmaals.

The best way to get around this is to switch off temporarily the firewall or anti-virus program, send the form once more and then switch the firewall or anti-virus program back on again. The problem doesn't originate in the browser settings, as the error message suggests. If sending the form was successful, a confirmation message is displayed.

The popular program "Norton Internet Security" allows switching off the "Privacy" option. Don't forget to switch it back on again!

If sending the form still fails, please send in the requested information by regular email to evaluatie@pellegrina.net.
